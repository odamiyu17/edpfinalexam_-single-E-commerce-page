# 🛒 E-Commerce React Project (Exam)

This project is a simple E-commerce web application built using **React + Vite**.  
Users can select products, add them to cart, and view their order summary.

## ⚙️ Requirements
Before running the project, make sure you have:
- Node.js installed (https://nodejs.org)
- Git installed

---

## 📥 How to Clone the Project

Open your terminal and run:

```bash
git clone https://github.com/odamiyu17/edpfinalexam_-single-E-commerce-page.git


Then go inside the project folder:

cd edpfinalexam_-single-E-commerce-page

🚀 How to Setup and Run (Vite + React)

Install dependencies: 

npm install

Run the development server:

npm run dev

Then open the link shown in the terminal (usually):

http://localhost:5173/
